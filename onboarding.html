What is Plex?

Plex is a media server + apps. Hyperdrive hosts the library; your devices stream from it. Think “your own Netflix,” except the catalog is community‑curated and requests are handled through Jellyseerr.

You’ll sign into Plex once, then switch to the Hyperdrive server inside the app.

Quick start
  1. Create/Sign in to a free Plex account at app.plex.tv.
  2. Install Plex apps on your devices (download links).
  3. In the app, switch to the Hyperdrive server. If you can’t see it, ask for access.
  4. Set streaming quality (below) so things play smoothly when you’re away from home.
  5. Request movies/series via Jellyseerr. Done.
Requesting content (Jellyseerr)

Go to jellyseerr.cartersanderson.com and Sign in with Plex. Search for a title → click Request. I prioritize 1080p movies and 720p/1080p TV for reliable remote playback. 4K is case‑by‑case.

New items appear in Plex automatically once they’re fetched, verified, and indexed. If something seems stuck, ping support (card below) with the title.

Recommended streaming quality
  • Wi‑Fi (Home/Good Internet): 1080p @ 8–10 Mbps
  • Cellular / Flaky Wi‑Fi: 720p @ 4–6 Mbps
  • Allow Direct Play / Direct Stream: On (if your device supports the format)

In the player, avoid “Original” for remote sessions—pick a fixed bitrate. If you see buffering, drop one step. Quality > bragging rights.

Subtitles & audio

Set your default in the app: Subtitles → Only Forced (or your language). Prefer SRT where available. Report out‑of‑sync or wrong subs and I’ll swap them.

4K & HDR expectations

Remote 4K (especially HDR + TrueHD/Atmos) can be rough. Use a capable device on strong Wi‑Fi, or request a 1080p version for smooth remote streaming. Local network playback is best for full‑fat 4K.

Troubleshooting in 60 seconds
  1. Lower Remote Quality one step (e.g., 10 → 6 Mbps).
  2. Switch networks (cellular ↔︎ Wi‑Fi) or move closer to the router.
  3. Force‑quit and relaunch the Plex app.
  4. Try another title. If only one fails, report that specific item.
  5. Still broken? See Support with device, app version, network, and time.
Etiquette & limits
  • No account sharing. Ask if a household member needs access.
  • Request in reasonable batches. Don’t carpet‑bomb 50 titles at once.
  • Downloads for travel are fine; don’t try to mirror the whole library offline.
How requests get fulfilled

Jellyseerr → Radarr/Sonarr → verified sources → renamed/organized → Plex library. New items show up under Recently Added. If a request lingers more than expected, it’s usually waiting on a good source or finishing processing.

Privacy

Playback data (what/when/bitrate) is used only to keep the system healthy and fix issues. If you want your activity hidden from shared dashboards, ask and I’ll adjust your visibility.
